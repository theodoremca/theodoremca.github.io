Ki67 Counter

Ki67 Counter - Quantifying nonhistone nuclear protein

Ki67 Counter Web Application. This cutting-edge tool is designed to analyze cellular proliferation in histological images, particularly calibrated for low-grade meningiomas at 10X magnification.

Functionality

  • Cell Counting

    Accurately counts cells within histological images and Accurately counts cells within histological images

  • Real-time Feedback

    Offers real-time feedback on the counting process, enhancing the user experience.

  • Image Management

    Implements a clean-up mechanism to delete old images, ensuring efficient storage management.

How to use?

Features

  • Dynamic Image Analysis

    Harnessing the computational might of MATLAB through Python, the application dynamically analyzes tissue samples. Tailored for low-grade meningiomas at 10X magnification.

  • FastAPI Integration

    Seamless communication between the frontend and the MATLAB-powered backend using FastAPI. Enables user-friendly image uploads and real-time processing.

  • Responsive Frontend

    Modern, responsive user interface built with Vue.js, TypeScript, and Tailwind CSS. Ideal for pathologists and researchers seeking an intuitive experience.
